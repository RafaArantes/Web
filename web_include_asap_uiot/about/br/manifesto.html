O que é o UIoT?

O UIoT é um middleware de código aberto dedicado ao gerenciamento de serviços de ambientes de internet das coisas foi criado com o intuito de ser capaz de controlar de forma transparente qualquer componente.

Porque UIoT?

O UIoT oferece uma API padronizada com formas intuitivas e simplificadas para adicionar novos dispositivos. UIoT é simples, seguro e de fácil de implementação. A internet das coisas e as coisas incríveis que ele pode realizar já são conhecidas por muitas pessoas.
