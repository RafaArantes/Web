Codificando pequenos dispositivos

Nós integramos novos dispositivos em nossa arquitetura de uma forma simples. Construindo diferentes soluções para diferentes tecnologias, escrever uma solução de uso genérico para cada sensor, depois disso, envolve a dinâmica de conversas abstratas com um sistema de mensagens iterativo.

conversas constantes com trocas de informação constrói uma aprendizagem para o dispositivo com base nos dados recolhidos da rede. Os dispositivos se comunicam entre si com nosso próprio protocolo. Já são mais de 30 casos de uso e pronto para o suportar mais de 500 dispositivos diferentes.
