Connected by nature.

UIoT connects everything based on UPnP, discovering all devices which are able to communicate and use then to create an IoT network.



What is UIoT?

UIoT is an open-source middleware dedicated to managing services in Internet-of-Things environments. It is designed to transparently allow control of any component able to communicate information.

Why is UIoT?

UIoT offers a standardized API with intuitive and simplified ways to add new devices. UIoT is simple, secure and easy to deploy. IoT and the amazing things it can achieve are already known by a lot of people.

Built for innovation

UIoT is more than a simple organization. UIoT was born to idealize a new paradigm of IoT. We're outstanding a new formulae of how a smart and connected world should be. You can across the world faster, with UIoT.

Involved in Layers

UIoT architecture represents the mastering of an IoT architecture.




Made by everyone

UIoT team is everything for us. They define our way of think, our process, our ideas and our goals. UIoT team coexists with a big focus in the future.

UIoT Service

The Service layer receives requests made by the Application layer, processes them so they can be sent to the other layers.

×

UIoT Communication

The Communication layer is responsible for receiving requests from the Service layer, and spread them through the UIoT network.

×

UIoT Execution

The Execution layer is responsible for receiving final commands from the UIoT network and executing them.

×

UIoT Application

The Application layer provides the Applets and Applications that will handle commands sent by the UIoT Network and start the process of execution. ×

UIoT Security

The Security layer makes sure that all the info flowing through the UIoT network is validated and trustworthy.

×

UIoT Quality

The Quality layer is responsible for making sure that no info is lost at any point during the flow of the UIoT Network.

×

UIoT Data

The Data layer stores, sorts and manages all the info that comes and goes inside the UIoT system.
